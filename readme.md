# Email Automation System

High-performance SMTP email automation with threading, template variables, and SMTP rotation.

## Features

- ✅ Send personalized emails to thousands of recipients
- ✅ Round-robin SMTP rotation for deliverability
- ✅ HTML email templates with dynamic variables
- ✅ Multi-threaded for high performance
- ✅ Automatic retry on connection failures
- ✅ Comprehensive logging
